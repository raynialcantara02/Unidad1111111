Orígenes de la Web

Web 2.0

La Web 2.0 se refiere a la segunda generación de la web, caracterizada por el aumento de la participación de los usuarios y la creación de contenido colaborativo. En lugar de ser una web estática, la Web 2.0 permite interacciones más dinámicas y ricas.

Web 3.0

La Web 3.0, a menudo llamada la web semántica, es una evolución que busca hacer que la web sea más inteligente mediante el uso de datos estructurados y tecnologías que faciliten la conexión y comprensión de la información de manera más eficiente.

La W3C y los estándares

La World Wide Web Consortium (W3C) es una comunidad internacional que desarrolla estándares para asegurar el crecimiento de la web de manera interoperable y accesible. Los estándares ayudan a garantizar que las tecnologías web funcionen correctamente en diferentes plataformas y dispositivos.

Lenguajes de etiquetas

SGML

El Standard Generalized Markup Language (SGML) es un estándar para definir lenguajes de marcado. HTML y XML son descendientes de SGML, que proporciona una estructura para la descripción de documentos.

XML

Extensible Markup Language (XML) es un lenguaje de marcado que define reglas para la codificación de documentos en un formato legible por humanos y máquinas. XML es utilizado para almacenar y transportar datos.

HTML

Hypertext Markup Language (HTML) es el lenguaje estándar para crear y diseñar páginas web. Define la estructura del contenido mediante el uso de etiquetas y atributos.

XHTML

Extensible Hypertext Markup Language (XHTML) es una versión de HTML que sigue las reglas de XML. Se diseñó para hacer que HTML sea más riguroso y compatible con los estándares XML.

HTML5

HTML5 es la quinta revisión del HTML y ofrece nuevas características y elementos para mejorar la funcionalidad y la presentación de las páginas web, incluyendo soporte para multimedia y gráficos.

Otros lenguajes relacionados al desarrollo web

CSS/CSS3

Cascading Style Sheets (CSS) es un lenguaje de hojas de estilo utilizado para describir la presentación de documentos escritos en HTML o XML. CSS3 es la última versión que incluye nuevas funcionalidades y mejoras.

JavaScript

JavaScript es un lenguaje de programación que permite la creación de contenido interactivo en las páginas web. Es fundamental para añadir dinámicas y funcionalidades avanzadas a los sitios web.

Ajax

Asynchronous JavaScript and XML (Ajax) es una técnica que permite actualizar partes de una página web sin recargarla completamente, mejorando la experiencia del usuario al hacer las aplicaciones web más rápidas y dinámicas.

Tecnologías y lenguajes del lado del servidor

Las tecnologías del lado del servidor, como PHP, Python, Ruby, y Node.js, se utilizan para crear aplicaciones web que procesan datos en el servidor antes de enviarlos al cliente. Son esenciales para la gestión de bases de datos y la lógica del negocio.

Navegadores

Los navegadores web son aplicaciones que permiten a los usuarios acceder y visualizar páginas web. Ejemplos incluyen Google Chrome, Mozilla Firefox, Safari e Internet Explorer.

Motores renderizados

Los motores de renderizado son componentes de los navegadores web que interpretan el HTML, CSS y JavaScript para mostrar las páginas web en la pantalla. Ejemplos incluyen WebKit, Blink y Gecko.

Herramientas de desarrollo

Software WYSIWYG

El software WYSIWYG (What You See Is What You Get) permite a los desarrolladores crear y editar páginas web mediante una interfaz gráfica, mostrando el resultado final en tiempo real.

Otras alternativas

Figma

Figma es una herramienta de diseño de interfaces y prototipos colaborativa basada en la web que permite a los diseñadores trabajar juntos en tiempo real.

InVision

InVision es una plataforma de diseño y colaboración que facilita la creación de prototipos interactivos y la retroalimentación en el diseño de interfaces.

Zeplin

Zeplin es una herramienta que proporciona una interfaz entre diseñadores y desarrolladores, facilitando la entrega de recursos y especificaciones del diseño.

GitHub

GitHub es una plataforma de alojamiento de código que utiliza el sistema de control de versiones Git, permitiendo la colaboración en proyectos de desarrollo web.

Inspección de código desde el navegador

La inspección de código desde el navegador permite a los desarrolladores examinar y depurar el código HTML, CSS y JavaScript directamente en el navegador web, facilitando el análisis y la corrección de errores.

El Diseño De Una Página Web

Tipos de páginas Web

Existen varios tipos de páginas web, incluyendo páginas estáticas, dinámicas, interactivas y de contenido generado por el usuario. Cada tipo tiene diferentes propósitos y características según las necesidades del proyecto.

Puntos básicos en la elaboración de una página Web

Al elaborar una página web, es fundamental considerar la estructura, el diseño, la usabilidad y la accesibilidad. Estos puntos aseguran que la página sea funcional y atractiva para los usuarios.

Estructura De Una Página Web

Jerarquía de directorio de un proyecto web

La jerarquía de directorios en un proyecto web organiza los archivos de manera que facilite el mantenimiento y la escalabilidad. Generalmente, incluye carpetas para HTML, CSS, JavaScript, imágenes y otros recursos.

Elementos de estructura para el diseño de una página Web: Concepto

Regla utilizada para validar y certificar una página web

La validación y certificación de una página web se realiza utilizando herramientas y estándares que aseguran que el código sea correcto y cumpla con las especificaciones establecidas por las organizaciones de estándares.

Reglas para escribir HTML

Escribir HTML de manera correcta implica seguir una serie de normas, como el uso adecuado de las etiquetas, la estructura jerárquica del contenido y el cumplimiento de los estándares para asegurar la interoperabilidad.

Estructura básica de un archivo web

La estructura básica de un archivo web incluye las secciones HTML, head y body. La sección <head> contiene metadatos y enlaces a archivos externos, mientras que la sección <body> contiene el contenido visible de la página.
